Draft Implementation

The aim of the scripts below is to scrape data from my local node and try to find a correlation between Tor and clearnet connections.

Script flow

  1. Run my own local node (Enable both Tor and clearnet network connections).
  2. Monitor and log addresses received from connected peers.
  3. Each peer returns around 1000 Ip Addresses
  4. Scrapes and analyze timestamps within the same peer:
    • Compare timestamps between Tor and clearnet addresses within the same peer responses and identifying matching Timestamps
